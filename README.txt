== Installation ==
1. Download the plugin from the WordPress repository or upload the plugin files to your `/wp-content/plugins/meteoart` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Add the widget to your site using the Block Editor, a shortcode, or the Widgets screen.


== Easy Installation ==
There are several options for installation

### Block Editor

1. Open the Block Editor for the post or page where you want the widget.
2. Add the "MeteoArt Widget" block.
3. Customize the widget settings as desired.

### Shortcode

Insert `[meteo city="Paris" country="France"]` into your content, replacing `CityName` and `CountryName` with your location.

### Classic Widgets

1. Go to `Appearance > Widgets`.
2. Add the "MeteoArt Widget" to your sidebar.
3. Configure the settings and save.
